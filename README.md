Запись в салон красоты (ASP.NET Minimal API)

Приложение позволяет клиентам регистрироваться, авторизоваться и записываться к мастерам на услуги в удобное время.
В админской части можно управлять мастерами, услугами и расписанием.

Ключевые технологии:

ASP.NET Core Minimal API

PostgreSQL в качестве базы данных

Entity Framework Core (Code-First)

Swagger для документирования API

NLog для логирования

Валидация телефонных номеров через стороннее API

Background Jobs для периодических задач и асинхронной обработки

Архитектура: слой доступа к данным, сервисный слой, эндпойнты Minimal API

Поддержка аутентификации и авторизации (JWT)
